int eaf_time_addclock_ext ( _Out_ eaf_clock_time_t *  dst,
_In_ const eaf_clock_time_t *  src,
_In_ const eaf_clock_time_t *  dif,
int  flags 
)

Add ‘src’ with ‘dif’ and store the result into ‘dst’.

By default, the ‘dst’ is left untouched if overflow will happen. But if ‘flags’ contains #EAF_TIME_IGNORE_OVERFLOW, the ‘dst’ still store the result, and -1 is returned.

Note
It is safe if ‘dst’ and ‘src’ point to the same location.
Parameters
[in]dstDestination
[in]srcSource
[in]difThe time need to add
[in]flagszero or following values:
  • #EAF_TIME_IGNORE_OVERFLOW
Returns
0 if overflow not happen, -1 if overflow occur
