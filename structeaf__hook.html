◆ on_message_handle_before

int(* on_message_handle_before) (_In_ uint32_t from, _In_ uint32_t to, _Inout_ eaf_msg_t *msg)

Hook a service is going to handle message.

This hook is called when a service is going to handle a request/response. If a non-zero code is returned, the service will not handle this message.

Warning
EAF do NOT automatically generate a response for any return code.
