Detailed Description

eaf_map_low is a sorted associative container that contains data structures with unique keys. Keys are sorted by using the comparison function Compare. Search, removal, and insertion operations have logarithmic complexity. Maps are usually implemented as red-black trees. Low level operation are available for advanced usage.
