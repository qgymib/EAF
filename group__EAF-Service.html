  arg 
)    EAF_COROUTINE_YIELD(fn, arg, EAF_COROUTINE_YIELD_TOKEN)

The extended version of eaf_yield.

Like eaf_yield, service will suspend until eaf_resume is called, and when service was suspended, fn will be called immediately. fn is a function with protocol eaf_yield_hook_fn.

Parameters
[in]fnCallback
[in]argUser defined argument passed to callback

◆ eaf_load()

int eaf_load ( void  )

Load EAF.

At this stage, EAF will try to initialize every registered services. If a service failed to load, other service in the same group will be exited. After this call return, EAF guarantee every registered services was either initialized or exited.

Returns
eaf_errno
