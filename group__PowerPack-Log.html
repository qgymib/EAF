◆ eaf_log_set_callback()

void eaf_log_set_callback ( _In_opt_ eaf_log_callback_fn  fn,
_Inout_opt_ void *  arg 
)

Set log callback.

Once set, all logs will be redirect to this callback. So if you still need output log on your screen, you need to do it manually.

Note
The callback does not affected by log level filter
You cannot set more than one callback, otherwise the later one will replace the earlier one.
Parameters
[in]fnCallback function
[in]argUser defined argument
