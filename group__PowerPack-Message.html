A macro to help send request and parse response.

Use ‘_0’ to access request in ‘code_serialize’, use ‘_1’ to access response in ‘code_deserialize’.

Note
This macro require Message Service is running.
Warning
After this macro, context will be invalid.
‘_0’ is only valid in ‘code_serialize’, ‘_1’ is only valid in ‘code_deserialize’.
